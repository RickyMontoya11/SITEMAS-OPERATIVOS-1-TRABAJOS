# DEFINICIONES 

**CARGA**
* El cargador sitúa él módulo de carga en la memoria principal, comenzando en la posición X. En general, se pueden aplicar tres métodos.
  * Carga absoluta
  * Carga reubicable
  * Carga dinámica en tiempo de ejecución
